Atomic Level: Atom

Reasoning:
- Atoms are the smallest, indivisible UI building blocks (buttons, inputs, labels).
- Place single-purpose, highly reusable components here.

Usage:
- Create one folder per component, e.g. `Button/` with `Button.tsx`, `Button.css`.
