Atomic Level: Molecule

Reasoning:
- Molecules are composed of multiple atoms working together (e.g., form row combining label + input + hint).
- Use for small, reusable groups of atoms that form a distinct UI unit.

Usage:
- Create one folder per molecule, e.g. `FormRow/` with `FormRow.tsx`, `FormRow.css`.

Policy note:
- Follow BEM naming and semantic HTML as defined in .github/agents/design-system-architect-agent.agent.md
